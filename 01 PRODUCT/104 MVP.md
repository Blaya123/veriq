# MVP Specification

## Purpose
Define the minimum viable product for Phase 1 launch.

## MVP Core Feature: Unified Inbox

The Unified Inbox is the heart of the MVP. It brings all customer messages from every channel into a single, intelligent timeline — with AI handling the heavy lifting.

### What It Does
- All customer messages appear in one unified timeline
- AI auto-responds to common queries
- AI qualifies leads based on conversation context
- AI books appointments when buying intent is detected

### Supported Channels (MVP)
| Channel | Type | Integration Method |
|---------|------|--------------------|
| WhatsApp | Messaging | WhatsApp Business API / Cloud API |
| Instagram | Social DM | Instagram Graph API |
| Facebook | Social DM | Facebook Messenger API |
| Telegram | Messaging | Telegram Bot API |
| Email | Communication | IMAP/SMTP |
| Website Chat | Web | Embeddable chat widget |

### Key User Flows

#### Flow 1: Sign Up & Onboard
1. User visits VERIQ website
2. Signs up with email and password
3. Creates workspace (name, industry, size)
4. Guided onboarding: connect a channel
5. Lands in empty inbox with "waiting for messages" state

#### Flow 2: Connect a Channel
1. User navigates to Integrations settings
2. Selects a channel (e.g., WhatsApp)
3. OAuth flow / API key entry
4. VERIQ syncs recent conversations
5. Channel appears as "Connected" with status indicator

#### Flow 3: Receive & Reply to Messages
1. Customer sends message (any channel)
2. Message appears in Unified Inbox in real-time
3. AI analyzes message and suggests response
4. User reviews, edits (optional), and sends
5. Reply is sent through the original channel

#### Flow 4: AI Lead Qualification
1. Customer asks a pricing/purchase question
2. AI detects buying intent
3. AI qualifies lead (budget, timeline, needs)
4. Lead profile is enriched automatically
5. User is notified of qualified lead with summary

#### Flow 5: AI Appointment Booking
1. Customer expresses interest in a meeting
2. AI detects booking intent
3. AI suggests available time slots
4. Customer selects time
5. Calendar event created; both parties notified

## Out of Scope (MVP)
- Multi-workspace support
- Advanced analytics and reporting
- AI training on custom data
- Marketplace / third-party apps
- Enterprise features (SSO, RBAC, audit logs)
- Mobile apps (web-only MVP)
- Video/voice calling
- Native desktop apps

## Success Criteria
| Metric | Target |
|--------|--------|
| Active businesses | 1,000 |
| Messages processed via AI | 50,000+ |
| AI suggestion acceptance rate | > 70% |
| Time to first reply (AI-assisted) | < 30 seconds |
| Inbox load time | < 2 seconds |
| Integration failure rate | < 1% |
